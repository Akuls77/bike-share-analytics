# Bike Share Analytics (dbt + Dagster)

Analytics engineering project using:
- Snowflake (Data Warehouse)
- dbt Core (Transformations)
- Dagster (Orchestration)

## Objective
Build an end-to-end, locally runnable data pipeline to analyze public bike share data and generate business insights.

## Tech Stack
- Python
- Snowflake
- dbt Core
- Dagster

## Project Structure
- dagster/ : ingestion & orchestration
- dbt/     : transformations & tests
- data/    : raw CSV inputs
